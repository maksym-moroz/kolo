# Reference App

This is the current canonical app shell for the Kolo platform baseline.

Purpose:

- prove that the shared platform can build and run on Android and iOS
- provide the default integration path for shared capabilities
- act as the truth source for host wiring, release wiring, and observability wiring until more app shells exist

Current hosts:

- `androidApp/`: Android application host
- `iosApp/`: iOS application host

Related metadata:

- `app-definition.yaml`: reference shell identity and capability composition contract
